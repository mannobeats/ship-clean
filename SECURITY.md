# Security Policy

Ship Clean scans local source code and writes project-local configuration files.
It does not need access to remote services to run its core quality checks.

## Supported Versions

Security fixes are shipped for the latest published version.

## Reporting a Vulnerability

Please report security issues privately before opening a public issue.

Email: security@mannobeats.com

Include:

- the affected Ship Clean version
- the command that triggered the issue
- the operating system and package manager
- a minimal reproduction when possible

We will acknowledge valid reports and coordinate a fix before public disclosure.
